# Scriptytt

Collection of experimental scripts and automation utilities for common development workflows.

## Features

- File batch processing scripts
- Development environment setup automation
- Data transformation utilities
- CLI helper tools

## Tech Stack

Shell, Python, JavaScript

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/scriptytt.git
cd scriptytt
Clone the repository and run individual scripts.
```

### Usage

```bash
See individual script files for usage instructions.
```
